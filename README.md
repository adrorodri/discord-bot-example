#Discord bot template in Typescript with Node.js

###Requirements:

- node 10 or newer
- npm

##How to run it:

1. Replace <BOT_TOKEN> with your bot token
2. Install dependencies:
```npm install```
3. Build the bot with:
```npm run build```
4. Run the bot with:
```npm run bot```
5. Modify the bot at your needs :)
